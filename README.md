# esami_universita
Progetto di Laboratorio di Basi di Dati che realizza un database per la gestione degli esami universitari.

Il sistema è basato su PostgreSQL e include la progettazione concettuale (schema ER), logica e l’implementazione completa della base di dati, con viste, funzioni e trigger per garantire coerenza e vincoli applicativi.

È inoltre presente un’applicazione web sviluppata in PHP, HTML e CSS che consente l’interazione con il database da parte di utenti con ruoli diversi (segreteria, docenti e studenti).
